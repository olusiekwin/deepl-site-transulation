# Images Directory

Place your custom images here. The following images are expected:

## Required Images

1. **hero-cover.jpg** (or .png, .webp)
   - Hero section background cover image
   - Recommended size: 1920x1080px or larger
   - Should be professional capital advisory/finance related
   - Will be used as full-width background cover

2. **financial-planning.jpg**
   - Value proposition section image
   - Recommended size: 800x600px
   - Strategic financial planning imagery

3. **infrastructure-projects.jpg**
   - Sectors section image
   - Recommended size: 800x600px
   - Infrastructure and energy projects imagery

4. **global-finance.jpg**
   - Global reach section image
   - Recommended size: 1600x900px (16:9 aspect ratio)
   - Global financial markets imagery

## Image Guidelines

- Use high-quality, professional images
- Optimize images for web (compress before uploading)
- Recommended formats: WebP (best), JPG, or PNG
- Ensure images are relevant to capital advisory and finance
- Maintain aspect ratios as specified

## File Structure

```
public/
  images/
    hero-cover.jpg
    financial-planning.jpg
    infrastructure-projects.jpg
    global-finance.jpg
```

## Fallback

If images are not provided, the sections will display with gradient backgrounds as fallback.
